IP 주소란 무엇인가?
-IP 통신에 필요한 고유 주소를 말하며, 통신 기능이 제공되는 기기들이 서로 통신을 하기 위해 필요한 고유 번호
-네트워크상에서 컴퓨터들이 서로 데이터를 정확하게 주고받기 위해 사용하는 디지털 주소
-IP 주소를 표시하는 데에는 현재 IPv4와 IPv6 두 가지 체계가 주로 사용되며, 세계적 표준이 되지는 않아서 잘 알려져 있지는 않지만 다른 버전들도 존재


Live Server 로 실행시킬때 나오는 이 주소의 의미는 무엇인가?
VS Code 등에서 Live Server를 실행했을 때 웹 브라우저에 나타나는 주소는 일반적으로 `http://127.0.0.1:5500` (또는 `http://localhost:5500`) 형태.

1. http://` (통신 프로토콜)
웹 브라우저와 웹 서버가 데이터를 주고받기 위해 사용하는 통신 규칙입니다. "웹 페이지 문서를 주고받겠다"는 약속을 의미.

2. `127.0.0.1` 또는 `localhost` (로컬 IP 주소)
-가장 핵심적인 부분으로, 인터넷상의 다른 곳이 아닌 자기 자신(현재 사용 중인 내 컴퓨터)'을 가리키는 특별한 IP 주소-루프백 주소
-내 컴퓨터 안에서 가상의 서버를 열고 스스로 접속할 때 사용

3. `:5500` (포트 번호)
-컴퓨터는 동시에 여러 네트워크 프로그램(메신저, 게임, 웹 브라우저 등)을 실행할 수 있으므로, 데이터가 정확히 어떤 프로그램으로 전달되어야 하는지 구분하기 위해 포트 번호를 사용
-`5500`은 Live Server 확장 프로그램이 가상의 웹 서버를 띄울 때 기본적으로 사용하는 포트 번호
