[Confirmed] 제공된 문서에 기록된 HTML 태그 및 속성, CSS 선택자 및 속성에 대한 전수 조사 결과입니다.

## 1. HTML 태그 및 주요 속성

### 1.1 기본 구조 및 문서 설정
* [cite_start]`<!DOCTYPE html>`: 웹 브라우저에 처리할 HTML 문서 형식을 명시하는 문서형 정의(DTD)입니다[cite: 137].
* [cite_start]`<html>`: HTML 문서의 시작과 끝을 지정합니다[cite: 140]. [cite_start]`lang` 속성을 통해 텍스트 언어 정보를 지정할 수 있습니다[cite: 510]. [cite_start](예: `<html lang="ko">` [cite: 119])
* [cite_start]`<head>`: 웹 브라우저에 직접 노출되지 않는 문서 정보를 정의합니다[cite: 143, 144].
* [cite_start]`<meta>`: 문서의 메타데이터를 정의합니다[cite: 146]. [cite_start](예: `<meta charset="UTF-8">` [cite: 121])
* [cite_start]`<title>`: HTML 문서의 제목을 지정합니다[cite: 147].
* [cite_start]`<body>`: 웹 브라우저에 표시되는 모든 내용을 작성하는 영역입니다[cite: 148, 149].

### 1.2 텍스트 작성 및 서식
* [cite_start]`<h1>` ~ `<h6>`: 주제나 제목 성격의 텍스트를 작성할 때 사용합니다[cite: 196]. [cite_start](예: `<h1>Heading level 1</h1>` [cite: 199, 201])
* [cite_start]`<p>`: 텍스트로 문장 또는 단락을 구성할 때 사용합니다[cite: 215]. [cite_start](예: `<p>내용</p>` [cite: 216])
* [cite_start]`<br>`: 텍스트를 줄 바꿈할 때 사용합니다[cite: 223]. [cite_start](예: `<br>` [cite: 224])
* [cite_start]`<blockquote>`: 문단 단위의 인용문을 작성하며, `cite` 속성으로 출처 URL을 남깁니다[cite: 235, 236].
* [cite_start]`<q>`: 문장 단위의 짧은 인용문을 작성하며, `cite` 속성으로 출처 URL을 남깁니다[cite: 239, 240].
* [cite_start]`<ins>`: 추가할 텍스트를 표시합니다[cite: 243].
* [cite_start]`<del>`: 삭제할 텍스트를 표시합니다[cite: 244].
* [cite_start]`<sub>`: 아래 첨자를 표시합니다[cite: 252].
* [cite_start]`<sup>`: 위 첨자를 표시합니다[cite: 253].
* [cite_start]`<strong>`: 텍스트의 의미를 강조하며 굵게 표시합니다[cite: 315, 316].
* [cite_start]`<em>`: 텍스트의 의미를 강조하며 기울어져 표시됩니다[cite: 323, 324].

### 1.3 그룹 짓기 및 목록
* [cite_start]`<div>`: 블록 요소와 인라인 요소를 그룹으로 묶을 때 사용합니다[cite: 260, 261].
* [cite_start]`<span>`: 인라인 요소를 그룹으로 묶을 때 사용합니다[cite: 263, 264].
* [cite_start]`<ul>`: 비순서형 목록을 생성합니다[cite: 267, 268].
* [cite_start]`<ol>`: 순서형 목록을 생성합니다[cite: 277, 278].
* [cite_start]`<li>`: `<ul>` 또는 `<ol>` 태그 내부에서 목록 내용을 구성합니다[cite: 268, 277].
* [cite_start]`<dl>`: 정의형 목록을 생성합니다[cite: 285, 286].
* [cite_start]`<dt>`: 정의형 목록의 용어를 명시합니다[cite: 286, 288].
* [cite_start]`<dd>`: 정의형 목록의 용어 설명을 명시합니다[cite: 286, 289].

### 1.4 링크, 이미지, 멀티미디어
* [cite_start]`<a>`: 링크를 생성합니다[cite: 295, 296]. [cite_start]`href` (대상 경로 지정), `target` (연결 방식 지정), `title` (설명) 속성을 사용합니다[cite: 298, 299, 300].
* [cite_start]`<img>`: 이미지를 삽입합니다[cite: 302, 303]. [cite_start]`src` (이미지 경로), `alt` (이미지 설명) 속성이 필수입니다[cite: 305, 306].
* [cite_start]`<audio>`: 오디오 요소를 삽입하며 `src`, `controls` 속성을 사용합니다[cite: 461, 462, 463].
* [cite_start]`<video>`: 비디오 요소를 삽입하며 `src`, `controls` 속성을 사용합니다[cite: 464, 465, 466].
* [cite_start]`<source>`: `<audio>`, `<video>` 태그 내에서 리소스 경로(`src`)와 미디어 타입(`type`)을 명시합니다[cite: 468, 469, 471].

### 1.5 폼(Form) 구성
* [cite_start]`<form>`: 폼 양식을 의미하며 `action` (서버 URL), `method` (전송 방식: get/post) 속성을 사용합니다[cite: 331, 332, 335, 336].
* [cite_start]`<input>`: 한 줄짜리 입력 요소를 생성합니다[cite: 338, 339]. [cite_start]`type` (text, password, checkbox, radio, button 등), `name`, `value` 속성을 가집니다[cite: 341, 342, 343].
* [cite_start]`<label>`: 폼 요소에 이름을 붙이며 `for` 속성을 사용해 상호작용 요소의 `id`와 연결할 수 있습니다[cite: 345, 346, 349].
* [cite_start]`<fieldset>`: 관련 있는 폼 요소끼리 그룹을 짓습니다[cite: 357, 358].
* [cite_start]`<legend>`: 그룹 지은 폼 요소에 이름을 붙입니다[cite: 357, 359].
* [cite_start]`<textarea>`: 여러 줄 입력 요소를 생성합니다[cite: 367, 368].
* [cite_start]`<select>`: 콤보박스를 생성합니다[cite: 376, 377]. [cite_start]`size`, `multiple` 속성을 가집니다[cite: 388, 389].
* [cite_start]`<option>`: 콤보박스에 항목을 추가하며 `value`, `selected` 속성을 가집니다[cite: 378, 382, 390].
* [cite_start]`<optgroup>`: `<option>` 항목들을 그룹으로 묶으며 `label` 속성으로 이름을 부여합니다[cite: 379, 381].
* [cite_start]`<button>`: 버튼 요소를 생성하며 `type` 속성을 지정합니다[cite: 392, 393, 394].
* [cite_start]추가 폼 속성: `disabled` (비활성화), `readonly` (읽기 전용), `maxlength` (글자 수 제한), `checked` (선택된 상태), `placeholder` (입력 힌트)[cite: 404, 406, 408, 412, 414].

### 1.6 표(Table) 만들기
* [cite_start]`<table>`: 표를 생성합니다[cite: 418, 419].
* [cite_start]`<caption>`: 표의 제목을 지정합니다[cite: 421, 422].
* [cite_start]`<tr>`: 표의 행을 생성합니다[cite: 427, 428].
* `<td>` / `<th>`: 표의 열을 생성합니다. [cite_start]`<th>`는 제목 셀에 사용하며 `scope` 속성(인식 범위)을 적용할 수 있습니다[cite: 432, 433, 456, 457].
* [cite_start]`<thead>` / `<tbody>` / `<tfoot>`: 표의 행을 그룹화합니다[cite: 444, 446].
* [cite_start]`<col>` / `<colgroup>`: 표의 열을 그룹화하며 `span` 속성으로 열의 개수를 지정합니다[cite: 448, 450, 452].
* [cite_start]셀 병합 속성: `rowspan` (행 병합), `colspan` (열 병합) 속성을 `<td>`나 `<th>`에 사용합니다[cite: 442, 443].

### 1.7 시맨틱 태그 및 글로벌 속성
* [cite_start]시맨틱 태그: 태그 이름만으로 역할과 의미가 명확한 태그입니다[cite: 477].
    * [cite_start]`<header>` (헤더 영역), `<nav>` (링크 영역), `<section>` (관련 내용 영역), `<article>` (독립적인 영역), `<aside>` (측면 영역), `<footer>` (푸터 영역), `<main>` (주요 내용 영역)[cite: 479, 480, 485, 486, 488, 489, 491, 492, 496, 497, 499, 500, 501, 502].
* [cite_start]글로벌 속성: 모든 태그에서 공통으로 사용 가능한 속성입니다[cite: 505].
    * [cite_start]`class` (클래스 지정), `id` (아이디 지정), `style` (인라인 스타일), `title` (추가 정보), `lang` (언어 정보), `hidden` (화면 숨김), `data-*` (사용자 정의 속성)[cite: 506, 507, 508, 509, 510, 511, 512].
